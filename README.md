# To-Do List App

A simple and clean to-do list web application built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Simple and user-friendly UI
- Task persistence using browser localStorage

## Tech Stack
- HTML
- CSS
- JavaScript
